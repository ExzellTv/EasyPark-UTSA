**EasyPark UTSA** Parking made easier at UT San Antonio.
This app is a smart parking management platform built for the UTSA community.  
It helps students, faculty, and visitors quickly find, reserve, and manage parking spaces on campus with ease.

## ✨ Features
- **Real-time availability** – see open spots instantly.
- **Smart occupancy detection** – optional OpenCV-based camera integration.
- **Secure payments & permits** – built-in ticketing and payment support. (TBD)
- **Open API** – integrate with other UTSA apps or third-party services. (TBD)

## 🛠 Tech Stack
- **Backend:** Python (FastAPI) 
- **Computer Vision (optional):** OpenCV
- **Database:** PostgreSQL / MySQL
- **Frontend / Mobile:** (to be decided) 
- **Deployment:** Docker + Cloud Hosting
