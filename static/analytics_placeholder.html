How to Edit Parking Masks

  • Run python3 tools/mask_creator.py.
  • Left-click to mark corners of each spot.
  • Right-click to complete a polygon.
  • Press S to save or D to delete a selected mask.
  • Spots are saved in data/spots.json with frame dimensions for auto-alignment.

After saving, restart your FastAPI server to see updates in the live feed.
